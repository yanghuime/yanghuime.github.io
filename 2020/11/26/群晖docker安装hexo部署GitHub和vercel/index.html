群晖docker安装hexo部署GitHub和vercel

首先，为什么要出这么一个docker版hexo？难道在电脑上安装不香吗？好吧，是因为我懒，因为docker轻松部署hexo，具有很高的便携性，在群晖上可以直接部署，可以让我随时随地都可以快捷方便的发表博客（这也才写的第一篇…）好吧，接下来正式开始：

1.准备

  • docker-hexo-镜像：kinghtxg/kinght-centos-hexo
  • GitHub
  • vercel（GitHub的账号不可以是qq邮箱注册，后续可以换绑）

2.开始

本篇文章主讲关于群晖docker搭建hexo，其他系统的docker请自行百度，步骤其实是差不多的

(1).群晖操作

打开docker-映像-新增-从URL添加：
https://registry.hub.docker.com/r/kinghtxg/kinght-centos-hexo/

(2).设置安装路径

双击启动-高级设置-映射文件夹（docker/hexo）装载路径（/blog）

(3).设置端口

(4).启动镜像

打开终端机，输入以下步骤查看hexo是否搭建完成。

(5).GitHub注册和创建仓库

（不建议用qq邮箱注册，因为后面vercel注册不了）自行百度创建仓库

(6).绑定ssh密钥

群晖docker终端机操作,开始绑定ssh密钥（毕竟方便嘛）

复制密钥，到GitHub保存

(7).hexo部署

修改hexo文件夹_config.yml填写仓库ssh地址，开始部署，会发现部署不了，是因为缺了部署插件，所以我们安装插件,重新上传。

(8).vercel部署

GitHub登陆连接后，导入仓库，填入hexo-GitHub的http地址，然后生成vercel仓库，会得到一个链接。
